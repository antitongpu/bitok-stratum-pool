Getting Started

How to mine Bitok on this pool

1. Get a Bitok Wallet

Download the Bitok wallet from the official website or compile from source. Generate a new address to receive your mining rewards.


Visit Bitok Explorer

2. Download Mining Software

Bitok uses the Yespower algorithm optimized for CPU mining. Recommended miners:


  • cpuminer-opt (recommended)
  • cpuminer-multi

3. Configure and Start Mining

Run the miner with the following command:


./cpuminer -a yespower --param-n=2048 --param-r=32 --param-key="BitokPoW" -o stratum+tcp://localhost:3032 -u YOUR_BITOK_ADDRESS.worker1

Parameters:

  • -a yespower - The mining algorithm
  • --param-n=2048 - Yespower N parameter
  • --param-r=32 - Yespower R parameter
  • --param-key="BitokPoW" - BitokPoW personalization key
  • -o stratum+tcp://... - Pool stratum address
  • -u ADDRESS.worker - Your Bitok address followed by worker name

4. Monitor Your Stats

Enter your Bitok address in the search box on the homepage or miners page to view your hashrate, workers, and payment history.
